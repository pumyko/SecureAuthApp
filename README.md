# SecureAuthApp
A training project on the implementation of a secure authentication system.

## Stack:
- **Backend:** Python 3.12, Flask
- **Security:** Flask-Limiter (Rate Limiting), Werkzeug (Password Hashing)
- **Infrastructure:** Docker, Nginx (Reverse Proxy), SSL (Self-signed)
- **CI/CD:** GitHub Actions (Bandit, Safety, Pytest)

## Run:
`docker-compose up --build`
